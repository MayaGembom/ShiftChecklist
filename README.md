#  <p align="center"> Welcome to ShiftChecklistApp </p>
</br>

## :handshake: About
>ShiftChecklist, is an Hebrew native **Shared Assessments Lists Application** between 3 different End-users: Owner, ShiftManager and Employee.
>The app was built to give a solution to the old fashion way of writing restaurant checklists on paper.
>ShiftChecklist will help you keep track of which assignments are done in the specific shift.
>Owner can **create** and **share** to ShiftManager / ShiftManager can **create** and **share** to Employee a list of assessments and always be sure that the list is showing **real-time data**.

## :sparkles: Features
> - **Login and Sign-up** with any valid phone number.
> - Every user have his own profile page & side bar. Side bar is determined by user type (Owner/ShiftManager/Employee).
> - Chose/Change your **profile picture**.
> - Each End-user have a different interface for his own permissions & options.
> - Employee/ShiftManager track assignments and do them.
> - Send shift assessments report to the owner email address.
